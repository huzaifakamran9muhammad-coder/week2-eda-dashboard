# 📊 Week 2 EDA Dashboard (Streamlit)

## 📌 Project Overview
This project is an **Exploratory Data Analysis (EDA) Dashboard** built using Streamlit.  
It allows users to explore, analyze, and visualize real-world datasets interactively.

The dashboard performs data cleaning, statistical analysis, and visualization to extract meaningful insights.

---

## 📂 Dataset Description
- The dataset used contains real-world structured data.
- It includes numerical and categorical features.
- Size: 1000+ records (recommended requirement met).

---

## 🧹 Data Cleaning Process
The following cleaning steps were applied:
- Removed duplicate records
- Handled missing values using mean imputation
- Ensured correct data types for analysis

---

## 📊 Exploratory Data Analysis (EDA)
The following analyses were performed:
- Dataset overview (shape, columns, data types)
- Statistical summary using describe()
- Correlation analysis using heatmap
- Distribution analysis using histograms
- Relationship analysis using scatter plots
- Outlier detection using box plots

---

## 📈 Features of Dashboard
- Interactive sidebar filters
- Column selection for analysis
- Dynamic visualizations
- Correlation heatmap
- Multiple chart types (Histogram, Boxplot, Scatter Plot)

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit
